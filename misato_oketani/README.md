# README

## ① ソフト名

オンライン図書管理アプリ（CLI版）

---

## ② 概要

- 本の検索ができる
- 貸出状況を確認できる
- 本を予約できる
- CLIで操作する
- データはローカルファイルに保存する

---

## ③ 主な機能

### ① 本検索
- タイトル検索
- 著者検索
- 本一覧表示

### ② 貸出確認
- 貸出状況表示
- 返却予定日表示

### ③ 予約機能
- 本を予約できる
- 予約情報を保存する

### ④ ユーザー機能
- ユーザー登録
- ログイン
- 利用履歴確認

---

## ④ 使い方

### ① 起動
./main.sh

### ② ログイン
- ユーザーID入力
- パスワード入力

### ③ メニュー操作

1. 本を検索
2. 貸出状況確認
3. 本を予約
4. 利用履歴
5. ログアウト

---

## ⑤ 動作環境

- Linux
- Bash
- ターミナル（CLI）
