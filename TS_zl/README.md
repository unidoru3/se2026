##README.md
ソフトウェア名: ToDo list

機能
タスクの追加: コマンドラインから新しいタスクを即座に登録できます。

一覧表示: 現在保存されているタスクを番号付きのリストで確認できます。

タスクの完了処理: 番号を指定して、タスクを「完了」状態に変更できます。

自動保存: プログラムを終了しても、タスクはテキストファイルに自動的に保存されます。

使い方
ターミナルまたはGit Bashを開きます。

プログラムを実行します（例: python main.py または ./todo_app）。

メニューが表示されるので、数字を入力して操作を選んでください。

1: タスクを表示する

2: 新しいタスクを入力する

3: タスクを削除/完了する

4: 完了したタスクを表示

q: アプリを終了する
