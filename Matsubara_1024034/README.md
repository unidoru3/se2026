## Studytimer-CLI

## 主な機能
1. ユーザー管理: auth コマンドによるログイン・ログアウト機能。

2. タスク管理: 学習予定の追加、一覧表示、完了マークの付与。

3. ポモドーロタイマー: 25分間の集中時間をターミナル上で計測し、終了時に通知。

4. 統計レポート: 日別・週別の学習時間を集計してテキスト形式で表示。

## 使い方

## 1. ログイン
1. study auth login

## 2. 学習タスクの登録
1. study task add "アルゴリズムの復習" --priority 高

## 3. 学習タイマー（25分）の開始
1. study timer start

## 4. 学習状況の確認
1. study report summary
