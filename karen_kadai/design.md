## 1. 画面構成（出力モード）

1. **ダッシュボード表示 (List表示)**
    * 実行コマンド：python main.py list
    * 表示内容：未完了の課題一覧、締切が近いタスクの強調表示、進捗率の表示。
2. **課題登録プロンプト (Input表示)**
    * 実行コマンド：python main.py add
    * 表示内容：対話形式でのタイトル、科目、締切日、メモの入力受付。
3. **課題詳細・編集 (Update表示)**
    * 実行コマンド：python main.py edit [タスクID]
    * 表示内容：指定したIDの情報の表示と、ステータスの入力受付。

## 2. データ構造 (MariaDB)
1. **タスクID (task_id)**: 主キー (INT, AUTO_INCREMENT)。
2. **課題内容 (title, memo)**: タイトルやメモ (TEXT)。
3. **締切日時 (deadline)**: 提出期限 (DATE)。
4. **ステータス (status)**: 状態（0:未着手、1:進行中、2:完了）。

## 3. 処理の流れ
1. **入力**: コマンド引数、または対話型プロンプトによるデータ取得。
2. **確認**: 入力値のバリデーション。
3. **保存**: MariaDB へ保存。
4. **出力**: 処理結果や一覧表をターミナルに表示。

## 4. システム構成
* **UI**: ターミナル
* **App**: Python CLI (argparse)
* **DB**: MariaDB
