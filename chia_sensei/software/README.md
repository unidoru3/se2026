
# 英語上手

単語を登録して、クイズで復習する。

## 機能
- 単語の追加・編集・削除
- ファイルから一括取り込み
- ランダム出題のクイズ
- 履歴と平均正答率
  
## 使い方
```
python egjz.py add apple リンゴ
python egjz.py import toeic.txt
python egjz.py quiz
```
