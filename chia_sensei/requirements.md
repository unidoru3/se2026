# 英語上手

英単語の意味を覚えるための単語暗記CLIソフト。
登録して、クイズで復習する。

## 機能要件
1. 単語を追加する
   - 英語と日本語の両方を指定して追加
   - 片方だけ指定すると、残りを対話で聞く
2. ファイルから一括で取り込む（.txt または .csv）
3. 登録済みの単語を編集する
4. 単語を削除する
5. 単語の一覧を表示する
6. クイズモードで出題し、正誤を判定する
7. クイズ履歴の統計を表示する

## 非機能要件
1. CLIで動く
2. 内部データはCSVで保存
3. 日本語が文字化けしない（UTF-8）
4. Python標準ライブラリだけで動く
5. 各コマンドの処理は1秒以内
