# テスト計画

| テスト項目 | 入力・操作 | 期待する結果 | 実際の結果 | 判定 |
|---|---|---|---|---|
| タスク追加 | 「1」を入力し、タスク名を入力 | タスクが追加される | タスクが追加された | OK |
| タスク一覧表示 | 「2」を入力 | 登録済みタスクが表示される | タスク一覧が表示された | OK |
| タスク削除 | 「3」を入力し、番号を指定 | 指定したタスクが削除される | タスクが削除された | OK |
| 完了状態変更 | 「4」を入力し、番号を指定 | タスクの状態が切り替わる | 未完了→完了に変更された | OK |
| 不正入力 | 存在しない番号を入力 | エラーメッセージを表示する | エラーメッセージが表示された | OK |
