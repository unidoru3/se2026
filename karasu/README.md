karasu
課題管理アプリケーションです。

機能要件

課題を追加できる
課題を削除できる
課題の締め切りを設定できる
課題の完了状態を変更できる
課題一覧を表示できる

非機能要件

シンプル
使いやすい
初心者でも扱える
データを保存できる
勝手に保存してくれる


画面構成

ログイン画面：ユーザー名とパスワードを入力してログインする
課題一覧確認画面：登録された課題を一覧表示する。課題の完了状態進行状況を%で確認できる
課題追加画面：課題名と締切日を入力して追加できる


システム構成

フロントエンド：HTML / CSS
バックエンド：Python
データ保存：SQLite


処理の流れ

ユーザがログインする
課題一覧を表示する
課題を追加する
データを保存する


データ設計

ユーザー情報
課題情報
締切日情報
