# SportTrack CLI

日々の筋トレやランニングの記録を、ターミナルから素早く記録・確認できる軽量なフィットネストラッカーツールです。

## 🚀 主な機能
* **クイック記録**: `run` コマンドでトレーニング内容や走行距離を1行で即座に記録。
* **ステータス確認**: `status` コマンドで、これまでに記録したデータの一覧や簡易的な統計を表示。
* **軽量・高速**: データベースを使わず、ローカルのJSONファイルでデータを高速に管理。

## 🛠️ 使い方

### 1. 初期設定
最初に使用する際、以下のコマンドで設定ファイルとデータ保存領域を自動生成します。
```bash
python main.py config
